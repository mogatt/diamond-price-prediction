# Diamond Price Prediction

Проект машинного обучения для прогнозирования стоимости бриллиантов по их физическим и качественным характеристикам.

## Цель проекта

Построить модель регрессии, которая предсказывает стоимость бриллианта на основе его параметров.

## Что выполнено

В рамках проекта были выполнены:

- исследовательский анализ данных;
- обработка пропущенных значений;
- анализ числовых и категориальных признаков;
- кодирование категориальных признаков;
- масштабирование данных;
- обучение нескольких моделей регрессии;
- сравнение результатов моделей;
- обучение итоговой модели;
- оценка качества итоговой модели.

## Использованные модели

В проекте рассматриваются:

- K-Nearest Neighbors;
- Ridge Regression;
- Support Vector Regression.

## Метрика

Для оценки качества модели используется Mean Squared Logarithmic Error — MSLE.
